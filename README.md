# TPrinterInfo
Mit der Komponente TPrinterInfo können Sie alle relevanten Daten der installierten Drucker auf Ihrem System ermitteln und in Ihrem Programm anzeigen. Das sind folgende Properties stehen zur Verfügung :

Attributes
AveragePPM
cJobs
ColorPrinting
Comment
Datatype
DefaultPrinter
DefaultPrinterID
DefaultPriority
DriverName
Location
Memory
PaperSizes
PaperTrays
PaperHeight
PaperWidth
Parameters
PortName
PrinterCount
PrinterIndex
PrinterListName
PrinterName
PrinterState
PrinterStateString
PrintProcessor
Priority
SepFile
ServerName
ShareName
StartTime
SupportDuplex
UntilTime


Für die Druckaufträge im Spooler stehen die folgenden Properties zur Verfügung :

JobIndex
JobName
JobNameList
JobOwner
JobNumPages
JobSize
JobStatus
JobSubmitTime
